DEFAULT_ELEMENT_VISIBILITY

The default visibility of the elements when they are added to the window.

public const bool DEFAULT_ELEMENT_VISIBILITY = false

Field Value

bool

Remarks

This value should not be changed. Each time the user adds an element to the window, it will try to toggle the visibility of the element.

INTERVAL_BETWEEN_READS

Defines the interval of milliseconds between different read key of the console (used in the Freeze(ConsoleKey) method).

public const int INTERVAL_BETWEEN_READS = 10
