In this section, you will find the list of all the elements available in the library. They will be accompanied by a brief description and a link to their respective documentation page.

They are divided into three categories:

  • Passive: the default elements, which do not require any specific action from the user.
  • Interactive: like a menu or prompt, these elements require user input and interaction.
  • Animated: elements that are animated, like a progress bar.
Note

An article is available to help you create your own elements. You can find it here.
